# Rider shortcuts


Key | Description
-- | --
cmd + f4 | close the current tab
option + up arrow | select
shft + alt + L | show file in solution explorer
cmd + ctrl + 1 | show Terminal
shft + cmd + b | build
shft + alt + f5 | run, re-run
cmd + t, y | run unit tests
shft + f5 | stop running
alt + f5 | debug - start
shft + f5 | debug - stop
