#### Settings
##### push only the current branch
```
git config --global push.default current
```

##### store git credential
```
git config credential.helper store
```

##### create a new branch
```shell
git checkout -b feature-install-script
```

#### Branching

##### Delete a branch

```
git branch -d feature-api-documentation
```
